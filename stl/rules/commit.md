## commit 规范

注意：使用英文编写commit！

### 1.任务

每完成一个任务，我们称之为Feature，所以提交的commmit信息为：

模版：
```
Feature: title.

Detaile message.

Author: email.
```

for example:

```
Feature: reading vector iterator.

iterator....

Author: light-city@datawhale.com
```

### 2.Bug修复

代码有问题，或者要对别人的代码添加patch，可以使用下面模版。

```
Fix: title.

Detaile message.

Author: email.
```

for example:

```
Fix: lost data for vector main.

iterator....

Author: light-city@datawhale.com
```
